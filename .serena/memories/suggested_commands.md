# Suggested Commands

## Development

- `yarn dev`: Start the development server (usually on http://localhost:3000).
- `yarn build`: Build the application for production.
- `yarn start`: Start the production server.

## Code Quality

- `yarn lint`: Run ESLint to catch code issues.

## File System (Darwin/macOS)

- Standard Unix commands available: `ls`, `cd`, `grep`, `find`, `cat`, etc.
