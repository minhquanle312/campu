Development commands:
- `yarn dev` or `next dev --port 3282` — run local dev server on port 3282
- `yarn build` — production build
- `yarn start` — run production server
- `yarn lint` — run ESLint
- `yarn format` — format TS/TSX files under src with Prettier
System is Darwin/macOS with zsh shell; standard utilities like `git`, `ls`, and `grep` are available, but project work should prefer dedicated IDE tools for file inspection and edits.
